Kronicle 6

A port to ES6 of the Kronicle SPA framework

Overview

A port of Kronicle SPA framework to ES6 as well as a rewrite of the main framework. Using http://addyosmani.com/largescalejavascript/ as inspiration.

High level goals:

  • Modular Design
    • Everything is a module, loaded by the core
  • Event Driven
    • Uses a pubsub pattern to communicate between moudles
  • Facade to present consistent simple API
